## Project: Lexi - AI English Speaking Tutor

A serverless web application built on AWS to help users practice English speaking with an AI tutor powered by Amazon Bedrock.

**Key Technologies:**
- Backend: Python, AWS Lambda, API Gateway, DynamoDB
- Frontend: Next.js 16, TypeScript, Tailwind CSS, Shadcn/ui
- AI/ML: Amazon Bedrock (Nova Micro), Transcribe, Polly
- Authentication: Amazon Cognito
- Hosting: AWS Amplify
