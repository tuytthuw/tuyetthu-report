### Lessons Learned:

* **Lesson 1: AWS SAM Simplifies Deployment** - AWS SAM makes defining and deploying serverless applications easier. template.yaml is single source of truth.

* **Lesson 2: DynamoDB Different from SQL** - DynamoDB is NoSQL, need different thinking about schema design. Partition key and sort key are very important for performance.

* **Lesson 3: Next.js App Router is Powerful** - App Router in Next.js 13+ provides flexible routing and Server Components, helping optimize performance.
