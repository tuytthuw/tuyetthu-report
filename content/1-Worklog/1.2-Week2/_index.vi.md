### Bài học:

* **Bài học 1: AWS SAM đơn giản hóa deployment** - AWS SAM giúp định nghĩa và deploy ứng dụng serverless dễ dàng hơn. Template.yaml là single source of truth.

* **Bài học 2: DynamoDB khác với SQL** - DynamoDB là NoSQL, cần suy nghĩ khác về schema design. Partition key và sort key rất quan trọng cho performance.

* **Bài học 3: Next.js App Router mạnh mẽ** - App Router của Next.js 13+ cung cấp routing linh hoạt và Server Components, giúp tối ưu performance.
