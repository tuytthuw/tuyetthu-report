## 1. Feelings about the program

The First Cloud AI Journey internship program has been a very practical and enriching learning experience. I felt challenged yet well-supported throughout the 8 weeks. The combination of theory, AWS labs, and building the Lexi product from start to finish helped me understand how a real web application comes together — from architecture design, user authentication, frontend/backend development to testing and deployment. The dedicated guidance from the AWS team and mentors was invaluable.

---

## 2. Satisfaction level

I am highly satisfied with the program (9/10). The curriculum is well-structured, covering important AWS services from basic to advanced levels. The weekly sessions, community events, and the process of building Lexi significantly enhanced the learning experience. The only area for improvement would be more hands-on time for complex frontend/backend integration tasks.

---

## 3. Knowledge gained

Throughout the program, I gained deep knowledge in:

- AWS core services: EC2, S3, Lambda, DynamoDB, Cognito
- AWS SAM and serverless deployment model
- Next.js App Router, React 19, Tailwind CSS v4, React Query
- Auth flow, route guards, token handling for web applications
- WebSocket for real-time speaking sessions
- Security best practices with IAM and least privilege principle

All this knowledge was directly applied to the Lexi project.

---

## 4. Skills improved

Beyond technical skills, I developed important soft skills:

- Writing well-structured and clear technical documentation
- Presenting system architecture and data flows more effectively
- Working efficiently in a team environment while integrating multiple components
- Managing time between studying, project development, and testing
- Solving problems under pressure when debugging cross-stack issues

These skills will be valuable for my future career.

---

## 5. Areas for improvement (for the program)

Some suggestions for the program's continued development:

1. Provide more hands-on time for complex frontend/backend integration tasks
2. Introduce project requirements and delivery milestones earlier
3. Organize more 1-on-1 mentoring sessions for challenging areas like authentication and real-time communication
4. Prepare detailed sample documentation for connecting backend outputs to frontend environment variables
5. Increase the frequency of peer review sessions for report content and demo preparation

---

## 6. Would you recommend the program to friends? Why?

Absolutely. I would recommend the FCJ internship program to anyone interested in cloud computing and web product development. The program offers:

- Real-world experience working with AWS
- Access to experienced mentors in the industry
- A supportive and active community
- A quality portfolio project like Lexi to add to your resume

This is truly an excellent launchpad for starting a career in cloud technology and full-stack development.
